## 🧠 What is HoyJob?

**HoyJob** is a **smart job hiring platform** that eliminates the chaos of traditional recruitment. Built for the modern era, it connects elite companies with verified top-tier specialists through an AI-powered matching engine — delivering curated, interview-ready candidates in under 48 hours.

We don't just post jobs. We engineer the perfect fit.

## 🏗️ Platform Architecture

```
HoyJob/
├── 🌐 index.html              ← Public landing page
├── 📁 pages/                  ← All platform views
│   ├── login.html             ← Unified login gateway
│   ├── signup.html            ← Agent & Recruiter registration
│   ├── admin-dashboard.html   ← Admin control center
│   ├── recruiter-dashboard.html ← Recruiter portal
│   ├── agent-dashboard.html   ← Agent portal
│   ├── marketplace.html       ← Job & talent marketplace
│   └── ...                    ← Terms, Privacy, Accessibility
├── 🎨 css/                    ← Styling & design system
├── ⚙️ js/                     ← Client-side logic & API calls
└── 🔧 backend/                ← Server, auth, and API layer
```

## 🚀 Core Features

### 🎯 AI-Powered Matching Engine
> Analyzes technical proficiency, historical project impact, and behavioral signals — scoring candidates for velocity, cultural resonance, and role longevity. You only see top-tier matches.

### 🔐 Dual-Portal System

| Portal | Role | Access |
|---|---|---|
| 👔 **Recruiter Portal** | Post jobs, review candidates, manage pipelines | `recruiter@gmail.com` |
| 🧑‍💼 **Agent Portal** | Build profiles, apply to roles, track applications | Register on signup |
| 🛡️ **Admin Dashboard** | Platform oversight, user management, analytics | `admin@gmail.com` |

### 🛍️ Talent Marketplace
> A curated open marketplace where verified agents showcase their skills, past projects, and availability — and recruiters browse talent on demand, without posting a single job listing.

### ✅ Verified Profiles
> Every agent profile is cross-referenced against authenticated project data. No resume inflation. No noise. Pure signal.

### 🌍 Global Reach
> Sourcing talent across **40+ countries** with localized compliance and bias-resistant algorithmic matching — merit always rises to the top.

### ⚙️ Smart Workflow
> Four-step hiring engine: **Precision Mapping → Vector Fit Score → Verified Signal → Optimized Interview Cycles**. From job definition to first interview in under **48 hours**.

## ⚙️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/Bloatyy/HoyJob.git

# 2. Navigate into the project
cd HoyJob

# 3. Install backend dependencies
cd backend && npm install

# 4. Start the backend server
node server.js

# 5. Open the frontend
# Open index.html in your browser or use a local server:
npx serve ..
```

> 🔗 Or just visit the live demo: **[hoyjob.onrender.com](https://hoyjob.onrender.com)**
